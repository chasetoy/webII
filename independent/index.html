Forms 101

HTML 1

This is the first step of creating the form section where you, the developer, opens the form element and sets the action to use your .php function which will handle the posting of the form and you set the method to "post" in order for the .php file to run correctly.



        <form action="testing2.php" method="post">

HTML 2

Now that you have the form element opened up we can go on and create the fields we would like to have included in the form. 



        <form action="testing2.php" method="post">

        First Name: <input type="text" name="first_name">

        Last Name: <input type="text" name="last_name">

        Email: <input type="text" name="email">

HTML 3

Now it is time to add the submit button and close the element, we set the name to "submit" [this is important when we write the PHP script] and the value to "Submit" [also important for the php script]. And now your form is finished.



        <form action="testing2.php" method="post">

        First Name: <input type="text" name="first_name">

        Last Name: <input type="text" name="last_name">

        Email: <input type="text" name="email">

        Message:<br><textarea rows="5" name="message" cols="30"></textarea>

	<input type="submit" name="submit" value="Submit">

	</form>
