Forms 101

Getting Started


We will begin by making THREE FILES:

  • index.html: this will be your webpage where the form can be hosted and filled out .
  • send.php: this is the script that will run when the form is submitted to send the email.
  • style.css: this will be used to style your website and the form as you see fit.


HTML 1


This is the first step of creating the form section where you, the developer, opens the form element and sets the action to use your .php function which will handle the posting of the form and you set the method to "post" in order for the .php file to run correctly.



        <form action="testing2.php" method="post">

HTML 2


Now that you have the form element opened up we can go on and create the fields we would like to have included in the form. 



        <form action="testing2.php" method="post">

          First Name: <input type="text" name="first_name">

          Last Name: <input type="text" name="last_name">

          Email: <input type="text" name="email">

HTML 3


Now it is time to add the submit button and close the element, we set the name to "submit" [this is important when we write the PHP script] and the value to "Submit" [also important for the php script]. And now your form is finished.



        <form action="testing2.php" method="post">

          First Name: <input type="text" name="first_name">

          Last Name: <input type="text" name="last_name">

          Email: <input type="text" name="email">

          Message:<br><textarea rows="5" name="message" cols="30"></textarea>

	  <input type="submit" name="submit" value="Submit">

	</form>

PHP 3


This is where we finish up by using the mail() function send two emails, one to the person that the forms belongs to, and another that sends a copy of the email to the person who is using the form. We just fill those function parameters with the variables that we want them to include. We then just issue an echo statement that lets the user know that the form has been submitted and their message has been sent and they will get a copy for confirmation and close up the php script with ?> 



        <?php

              if(isset($_POST['submit'])){

                $to = "example.email@somewhere.com"; // this is your Email address

                $from = $_POST['email']; // this is the sender's Email address

                $first_name = $_POST['first_name'];

                $last_name = $_POST['last_name'];

                $subject = "Form submission";

                $subject2 = "Copy of your form submission";

                $message = $first_name . " " . $last_name . " wrote the following:" . "\n\n" . $_POST['message'];

                $message2 = "Here is a copy of your message " . $first_name . "\n\n" . $_POST['message'];

                $headers = "From:" . $from;

                $headers2 = "From:" . $to;


                mail($to,$subject,$message,$from,$headers);

                mail($from,$subject2,$message2,$to,$headers2); // sends a copy of the message to the sender

                echo "Mail Sent. Thank you " . $first_name . ", we will contact you shortly.";

                // You can also use header('Location: thank_you.php'); to redirect to another page.

              }

        ?>
